# v0.4.0

### Added

- `confluence-sync pull <pageId|url> [--out <dir>] [--children]` — 역방향: Confluence 페이지/폴더를 읽어 `.md` 로 생성
  - **페이지·폴더(`/folder/<id>`) URL 모두 지원** — 폴더는 디렉토리로, 폴더/페이지 자식을 재귀 복원
  - `--children`: 하위까지 재귀 복원(자식 있는 페이지는 `<제목>/README.md` 폴더로 펼침)
  - 본문은 `export_view`(HTML)를 turndown 으로 변환 — 코드블록(언어 포함)·GFM 표 지원
  - 참조 이미지/첨부를 `.md` 옆에 내려받고 `![](파일명)` 으로 로컬화(외부 URL 이미지는 유지)

### Changed

- 런타임 의존성에 `turndown` 추가(HTML→Markdown 변환)

### Notes

- storage/export_view → Markdown 은 근사 변환(무손실 왕복 아님). 내부 페이지 링크는 현재 절대 URL로 유지
