# v0.5.0

### Added

- sync 시 **README 없는 폴더 → Confluence 폴더로 생성**하고 그 밑으로 문서를 동기화(기존엔 루트 직속). 중첩 폴더·멱등(삭제 시 재생성) 지원. `write:folder:confluence` 스코프 필요
- `--rebuild` 가 매핑된 폴더도 삭제(`delete:folder:confluence` 필요)

### Changed

- 부모 결정을 "컨테이너(README 페이지 또는 폴더)" 모델로 일반화 — README 있는 트리는 기존과 동일
- 환경변수 `CONFLUENCE_PARENT_PAGE_ID` → **`CONFLUENCE_PARENT_ID`**(페이지·폴더 id 모두 가능). 구 이름도 하위호환으로 계속 인식
- README 스코프 안내를 기능별 단일 표로 정리, `.env.example` 동기화
