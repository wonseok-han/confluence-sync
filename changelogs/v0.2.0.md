# v0.2.0

### Added

- `confluence-sync init` — 대화형으로 cwd 에 `.env` 생성(토큰 입력은 *로 가림, 기존 파일 덮어쓰기 가드, 비대화형은 빈 템플릿)
- `--help` / `-h`, `--version` / `-v` 옵션
- 컬러 출력(상태 라벨·요약·help·에러). `--no-color`/`NO_COLOR` 또는 비-TTY 에서 자동 비활성

### Changed

- 단일 파일(`sync.ts`)을 역할별 모듈로 분리: `config`·`markdown`·`docs`·`mapping`·`confluence`·`init`·`help` + 얇은 진입점
- README 설치 안내를 일반 사용자용으로 상세화(액세스 토큰 발급, `~/.npmrc` 스코프 매핑, 빠른 시작)
