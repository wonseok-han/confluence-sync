# v0.1.0

### Added

- Markdown 디렉토리 → Confluence Cloud 단방향 동기화 CLI (`confluence-sync`)
- 폴더 계층 동기화: 폴더의 `README.md`가 대표 페이지, 같은 폴더 문서는 자식으로 생성
- 변경 감지(제목+변환결과 해시)로 바뀐 문서만 갱신, 동일 문서는 스킵
- 내부 `.md` 링크 → 페이지 링크 변환, 로컬 이미지 → 첨부 업로드, 코드블록 → code 매크로
- 삭제된 페이지 자동 재생성 및 이를 가리키던 문서 링크 재연결
- 선택 동기화: 파일/폴더 위치 인자로 일부만 동기화(부모 README 자동 포함)
- `--base`/`--mapping`/`--force`/`--verify`/`--dry-run`/`--list`/`--rebuild` 옵션
- Git 호스팅 Package Registry 글로벌 설치 지원 (`@wonseok-han/confluence-sync`)
