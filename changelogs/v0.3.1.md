# v0.3.1

### Fixed

- 이미 같은 파일명 첨부가 있는 페이지를 재동기화할 때 이미지 업로드가 `400 Cannot add a new attachment with same file name` 으로 실패하던 문제 수정 — 기존 첨부가 있으면 생성 대신 **데이터만 새 버전으로 갱신**

### Changed

- 변경 감지 해시에 **참조 이미지의 내용 해시 포함** — 마크다운이 그대로여도 같은 파일명으로 이미지만 교체하면 해당 문서가 자동으로 갱신 대상이 됨(기존엔 `--force` 필요). 이미지 없는 문서의 해시는 기존과 동일
