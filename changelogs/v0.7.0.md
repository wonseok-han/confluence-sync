# v0.7.0

### Added

- `confluence-sync pull --space` — 대상 지정 없이 **스페이스(CONFLUENCE_SPACE_KEY) 전체**를 홈페이지부터 재귀적으로 가져오기
- pull 시 이미지를 각 문서 옆 **`attachments/<문서명>/` 하위**로 모아 저장(링크도 해당 경로로). 문서 폴더에 이미지가 섞이지 않음

### Fixed

- pull 재귀 시 **페이지 하위의 폴더**를 따라가지 않아 폴더로 묶인 문서가 대량 누락되던 문제 수정(페이지도 하위 페이지+폴더 모두 재귀)
- pull 중 특정 노드 조회 실패 시 전체가 중단되던 문제 — 해당 노드만 건너뛰고 계속 진행
- 한글·공백이 든 이미지/내부링크 경로가 퍼센트 인코딩되어 sync 시 파일명이 깨지던 문제 수정(경로 디코드). 공백 경로는 `![](<...>)` 로 안전하게 출력
- pull 한 코드블록이 전부 ```` ```java ```` 로 나오던 문제 수정 — Confluence 가 언어 미지정 시 `brush: java` 를 기본으로 렌더하므로, 이제 **`body.storage` 의 실제 language 파라미터**를 읽어 사용. 언어 미지정 블록은 ```` ```plaintext ```` 로 출력
- 코드블록의 줄 끝 공백(원본에 다수 존재)을 트림해 출력 정리
- Confluence 가 색상 텍스트용으로 끼워 넣는 인라인 `<style>` 의 CSS 가 본문(제목 포함)에 텍스트로 새어나오던 문제 수정 — `<style>`/`<script>` 등 제거
- 리스트 항목이 `<li><p>…</p></li>` 로 감싸져 항목마다 빈 줄이 들어가던 문제 수정(항목 사이를 붙임)
- 불필요한 백슬래시 이스케이프 완화 — 단어 내부 `\_`(예: `:11_eleven_square_blue:`), 줄 중간의 `\-` `\+` `\.` `\>`(예: `**7️⃣**\-**1️⃣**`), 링크가 아닌 `\[…\]`. 줄 맨 앞의 진짜 마커와 코드블록 내부는 보존

### Notes

- 첨부 이미지 다운로드에는 토큰 `read:attachment:confluence` 스코프가 필요(없으면 `![](파일명)` 링크만 생성). README 스코프 표에 반영
